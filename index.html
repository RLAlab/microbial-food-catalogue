Microbial Food Innovation Catalogue

An open-source catalogue of databases and computational tools to support the AI-driven acceleration of microbial food development.

This resource supports the AI Agenda for Microbial Foods. View on GitHub

About This Resource

Microbial foods offer a sustainable and nutritious alternative to traditional livestock, but their development faces hurdles in sensory performance, scale-up, and safety. Accelerating innovation requires leveraging Artificial Intelligence across the entire development pipeline.

This catalogue is a living resource designed to support researchers by providing a curated list of relevant databases and computational tools. It is intended to support the AI Agenda for Microbial Foods, fostering coordinated progress in food-tailored data, self-driving execution, predictive modelling, and evidence-based translation.

How to Contribute

This is an open-source, community-driven project. We welcome contributions to keep it accurate and comprehensive.

  1. Go to the project's GitHub repository.
  2. Navigate to the /data/ folder and choose either databases.csv or tools.csv.
  3. Click the pencil icon to edit the file directly on GitHub.
  4. Add your new resource as a new row, following the existing format.
  5. Propose your change by submitting a "Pull Request."
